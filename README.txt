Wine Makers Union Analytics Dashboard - A Power BI project that provides an in-depth analysis of a fictional wine producers' union, showcasing the types of grapes produced and various production trends among its members. Based on fictional data, this interactive dashboard allows users to explore key aspects of the union's operations and production patterns. Access the PDF overview of the project and download the remaining files to open in Power BI Desktop from this repository (note that the PDF does not include interactive features).
